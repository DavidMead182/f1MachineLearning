About

This web app has two main features:

1. Ability to view and create data dashboards for any formula race in the fastf1 api.

2. Ability to apply/see predictions from machine learning models.



I am currently a uni student, who is looking to create a webapp that can showcase my skills in web development, machine learning and data analysis.

My aims for this project are to:

1. Create a webapp that can be used by anyone to view and create data dashboards.

2. Create a webapp that is dynamic, I am using custom CSS over bootstrap or tailwind, so I can make a fully custom css toolkit for myself.

3. Create a webapp that can be used to showcase my machine learning skills.

4. Create a webapp that can be used by anyone to use my models to make predictions for different races, based on perimeters given

5. Create a webapp that can be used to showcase my data visualisation skills.

6. Create a fully functional website for others to create data dashboards.

Remember! This is a working progress ...
